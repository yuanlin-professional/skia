Skia 是一个开源的 2D 图形库 (2D Graphics Library)，提供了可在多种硬件和软件平台上运行的通用 API。它是 Google Chrome 和 ChromeOS、Android、Flutter、Mozilla Firefox 和 Firefox OS 以及许多其他产品的图形引擎 (Graphics Engine)。

Skia 由 Google 赞助和管理，但在 BSD 自由软件许可证 (BSD Free Software License) 下可供任何人使用。虽然核心组件的工程开发由 Skia 开发团队完成，但我们接受来自任何来源的贡献。

- 官方源代码仓库：
  [skia.googlesource.com/skia](https://skia.googlesource.com/skia)。
- 问题追踪器 (Issue Tracker)：[bug.skia.org](https://bug.skia.org/)。
- 讨论论坛：
  [skia-discuss@googlegroups.com](https://groups.google.com/forum/#!forum/skia-discuss)。
- [API 参考与概述](https://skia.org/docs/user/api/)。
- Skia Fiddle：[fiddle.skia.org](https://fiddle.skia.org/c/@skcanvas_paint)。

## 支持平台

- Windows 10、11
- macOS 11 或更高版本
- iOS 13 或更高版本
- Android 4.3 (JellyBean) 或更高版本
- Ubuntu 18.04+、Debian 10+、openSUSE 15.2+ 或 Fedora Linux 32+
