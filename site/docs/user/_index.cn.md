如果你想编写使用 Skia 库的代码，这里就是你需要的地方。

Skia 图形库可用于绘制文本、几何图形和图像：

  * 带透视的 3x3 矩阵 (3x3 matrices w/ perspective)
  * 抗锯齿 (antialiasing)、透明度 (transparency)、滤镜 (filters)
  * 着色器 (shaders)、混合模式 (xfermodes)、遮罩滤镜 (maskfilters)、路径效果 (patheffects)
  * 亚像素文本 (subpixel text)

Skia 目前支持的设备后端 (Device backends) 包括：

  * 光栅化 (Raster)
  * OpenGL
  * PDF
  * XPS
  * SVG
  * Picture（用于录制并在另一个 Canvas 中回放）
