Skia 高度依赖我们的单元测试和 GM 测试套件来进行正确性测试 (Correctness Testing)，这些测试由 DM 测试工具提供支持。测试由我们的 trybots 在每次提交时执行，覆盖我们支持的大部分平台和配置。
Skia [Gold](https://gold.skia.org) 是一个用于分类这些结果的 Web 界面。

我们还拥有一套完善的性能测试，由 nanobench 工具提供支持，并可通过 Skia [Perf](https://perf.skia.org) Web 界面访问。

Cluster Telemetry 是一个强大的框架，帮助我们在多达一百万个网站上捕获和基准测试 SKP 文件（一种绘制命令的二进制格式）。

有关我们各种测试工具的更多详细信息，请参阅各个子页面。
