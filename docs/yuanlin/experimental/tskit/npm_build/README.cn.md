# tskit/npm_build - npm 发布构建

## 概述

`npm_build/` 包含用于将 tskit 打包发布到 npm 的构建产物和示例文件。

## 目录结构

```
npm_build/
├── example.html     # 使用示例 HTML 页面
└── types/           # TypeScript 类型定义文件
```

## 关键文件

- **example.html**: 演示如何在网页中加载和使用 tskit 的示例
- **types/**: 发布到 npm 时包含的 `.d.ts` 类型定义文件

## 相关文档与参考

- npm 包配置: `../package.json`
- TypeScript 接口: `../interface/`
