# lottie-web-puppeteer - Lottie Puppeteer 捕获镜像

## 概述

基础的 Lottie-web Puppeteer 捕获环境 Docker 镜像，用于在 Chrome 中渲染和捕获 Lottie 动画帧。

## 目录结构

```
lottie-web-puppeteer/
└── Dockerfile   # Docker 镜像定义
```

## 依赖关系

- lottie-web JavaScript 库
- Puppeteer
- Chrome 浏览器
