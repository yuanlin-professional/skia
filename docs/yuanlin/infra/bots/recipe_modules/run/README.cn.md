# run - 命令运行模块

## 概述

`run` 模块提供命令执行的工具函数，封装了在不同平台上运行外部命令的细节。它支持超时控制、错误处理、输出捕获等功能。

## 目录结构

```
run/
├── __init__.py    # DEPS 依赖声明
├── api.py         # RunApi 核心类
└── examples/      # 使用示例和测试
```

## 关键文件

### api.py
提供命令执行的高级接口，包括：
- 带超时的命令执行
- 输出捕获和日志记录
- 跨平台命令适配
