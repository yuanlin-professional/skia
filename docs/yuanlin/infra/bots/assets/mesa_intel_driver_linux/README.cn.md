# mesa_intel_driver_linux - Linux Mesa Intel 驱动

## 概述

Linux 平台的 Mesa Intel GPU 驱动资源，用于在 Intel GPU 设备上运行 Skia 的 OpenGL/Vulkan 测试。

## 目录结构

```
mesa_intel_driver_linux/
├── create.py            # 自动化创建脚本
├── mesa-driver-builder/ # 驱动构建配置
├── README.md            # 原始说明文档
└── VERSION              # 当前版本号
```

## 依赖关系

- 被 Intel GPU Linux 测试任务使用
