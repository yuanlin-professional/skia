# mesa_intel_driver_linux_22 - Linux Mesa Intel 驱动 (v22)

## 概述

Linux 平台的 Mesa Intel GPU 驱动资源（版本 22 系列），为 Intel GPU 提供较新版本的 OpenGL/Vulkan 驱动支持。

## 目录结构

```
mesa_intel_driver_linux_22/
├── create.py            # 自动化创建脚本
├── mesa-driver-builder/ # 驱动构建配置
├── README.md            # 原始说明文档
└── VERSION              # 当前版本号
```

## 依赖关系

- 被需要较新 Mesa 版本的 Intel GPU 测试任务使用
