# dwritecore - DirectWrite Core 库

## 概述

Microsoft DirectWrite Core 库资源，用于 Windows 上的文本渲染测试。DWriteCore 是 DirectWrite 的跨版本兼容实现。

## 目录结构

```
dwritecore/
├── create.py   # 自动化创建脚本
└── VERSION     # 当前版本号
```

## 依赖关系

- 被 Windows 文本渲染相关测试任务使用
