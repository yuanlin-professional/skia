# bazelisk - Bazelisk 版本管理工具

## 概述

Bazelisk 是 Bazel 的版本管理包装器，自动下载并使用项目指定版本的 Bazel。此为通用版本资源。

## 目录结构

```
bazelisk/
├── README.md   # 原始说明文档
└── VERSION     # 当前版本号
```

## 依赖关系

- 另见各平台特定版本：`bazelisk_linux_amd64`、`bazelisk_mac_amd64` 等
