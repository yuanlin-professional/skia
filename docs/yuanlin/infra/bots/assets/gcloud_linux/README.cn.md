# gcloud_linux - Linux Google Cloud SDK

## 概述

Linux 平台的 Google Cloud SDK（gcloud）资源，提供与 Google Cloud 服务交互的命令行工具。

## 目录结构

```
gcloud_linux/
├── create.py   # 自动化创建脚本
└── VERSION     # 当前版本号
```

## 依赖关系

- 被需要访问 Google Cloud 服务的任务使用
