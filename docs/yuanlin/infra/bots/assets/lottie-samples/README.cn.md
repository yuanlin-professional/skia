# lottie-samples - Lottie 动画样本数据

## 概述

Lottie 动画格式的样本数据资源，用于 Skottie（Skia 的 Lottie 动画播放器）的测试和性能评测。

## 目录结构

```
lottie-samples/
├── create.py   # 自动化创建脚本
└── VERSION     # 当前版本号
```

## 依赖关系

- 被 Skottie 测试和性能评测任务使用
- 与 `infra/lottiecap/` 配合使用

- [Skottie](https://skia.org/docs/user/modules/skottie/)
