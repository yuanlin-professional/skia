# binutils_linux_x64 - Linux x64 GNU Binutils

## 概述

Linux x64 平台的 GNU Binutils 工具集资源，包含链接器、汇编器等二进制工具。

## 目录结构

```
binutils_linux_x64/
├── create.py   # 自动化创建脚本
└── VERSION     # 当前版本号
```

## 依赖关系

- 被 Linux 编译任务使用
- 与 `clang_linux` 编译器配合使用
