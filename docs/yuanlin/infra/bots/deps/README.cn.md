# deps - 依赖管理

## 概述

`deps/` 目录包含用于管理 Skia CI 系统依赖关系的 Go 源代码。这些文件负责生成和维护构建系统所需的外部依赖配置。

## 目录结构

```
deps/
├── deps.go       # 依赖定义
├── deps_gen.go   # 自动生成的依赖代码
└── generate.go   # 依赖生成逻辑
```

## 关键文件

### deps.go
定义 Skia CI 系统的外部依赖列表和版本信息。

### deps_gen.go
由 `generate.go` 自动生成的依赖代码，不应手动编辑。

### generate.go
包含依赖代码的生成逻辑，用于自动化更新依赖配置。

## 依赖关系

- 被 `gen_tasks_logic/` 在任务生成过程中引用
- 定义了 CIPD 包和其他外部工具的版本
