GrGlInterface 自动生成
============================

背景
----------

在高层次上，创建 GrGLInterface（一种与类 GL GPU 交互的通用方式）的前三个步骤是：

  - 组装 (Assemble)：将一组函数指针复制到结构体中
  - 验证 (Validate)：确保所宣告的函数指针确实存在
  - 能力检测 (Capabilities)：根据结构体中的函数计算哪些快速/慢速路径被启用（简称 GrGLCaps）

自动生成
--------------

前两个步骤已通过本文件夹中的基于表格的生成脚本实现自动化。该表格采用 JSON5 格式（类似 JSON，但支持注释）。

编辑完成后，可以通过在本文件夹中运行 `make generate` 来重新生成组装/验证代码。
