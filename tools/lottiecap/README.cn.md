在浏览器中捕获 Lottie 胶片条 (Filmstrip)
=======================================

命令行应用程序，用于在浏览器中从 Lottie 文件构建 5x5 胶片条，然后将该胶片条导出为 1000x1000 的 PNG 图像。

首先运行

    $ npm install

然后运行

    $ node ./lottiecap.js --input some_lottie_file.js

获取更多帮助：

    $ node ./lottiecap.js -h

需求
------------

你需要安装较新版本的 'node'，即 v8.9 或更高版本。从以下地址获取最新版本的 Node：

    https://nodejs.org/en/download/

你工作站的发行版不太可能有较新版本的 Node 可用。
