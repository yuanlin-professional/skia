# 如何贡献

我们非常欢迎您对本项目提交补丁和贡献。只需遵循以下几条简单的准则。

## 贡献者许可协议 (Contributor License Agreement)

向本项目的贡献必须附带贡献者许可协议 (Contributor License Agreement)。您（或您的雇主）保留对您贡献内容的版权；
此协议只是授予我们使用和重新分发您贡献内容的许可，作为项目的一部分。请访问 <https://cla.developers.google.com/>
查看您当前已签署的协议或签署新的协议。

通常您只需提交一次 CLA，因此如果您之前已经提交过（即使是为其他项目），您可能不需要再次提交。

## 代码审查 (Code Reviews)

所有提交，包括项目成员的提交，都需要经过审查。
请参阅代码贡献指南：https://skia.org/docs/dev/contrib/
